# PythonGS

Instrução de instalação:

Para utilizar o código, além da instalação do Python e do seu editor de código de preferência (VS Code, PyCharm etc.), deverão ser instaladas duas bibliotecas adicionais: oracledb e pandas.

Para instalar ambas as bibliotecas, abra o terminal no seu editor de código e execute os seguintes comandos:

```bash pip install``` 
```oracledb pip install pandas ```
